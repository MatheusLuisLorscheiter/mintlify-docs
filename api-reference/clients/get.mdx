---
title: "Buscar Cliente"
api: "GET /api/v1/clients/{id}"
description: "Busca um cliente específico pelo ID"
---

<Tip>
Retorna todos os detalhes de um cliente cadastrado no sistema.
</Tip>

<Warning>
Você precisa de uma API key válida no header x-api-key para usar este endpoint.
</Warning>

## Path Parameters

<ParamField path="id" type="string" required>
  ID único do cliente (UUID)
</ParamField>

## Headers

<ParamField header="x-api-key" type="string" required>
  API key de autenticação
</ParamField>

<RequestExample>

```bash cURL
curl -X GET "https://api.sistema-representantes.com.br/api/v1/clients/123e4567-e89b-12d3-a456-426614174000" \
  -H "x-api-key: sua-api-key-aqui"
```

```javascript JavaScript
const response = await fetch('https://api.sistema-representantes.com.br/api/v1/clients/123e4567-e89b-12d3-a456-426614174000', {
  method: 'GET',
  headers: {
    'x-api-key': 'sua-api-key-aqui'
  }
});

const result = await response.json();

if (result.success) {
  console.log('Cliente encontrado:', result.data);
} else {
  console.error('Erro:', result.error);
}
```

```python Python
import requests

url = "https://api.sistema-representantes.com.br/api/v1/clients/123e4567-e89b-12d3-a456-426614174000"
headers = {
    'x-api-key': 'sua-api-key-aqui'
}

response = requests.get(url, headers=headers)
result = response.json()

if response.status_code == 200:
    print(f"Cliente: {result['data']['nome']}")
    print(f"Email: {result['data']['email']}")
    print(f"Cidade: {result['data']['cidade']}")
    print(f"Status: {'Ativo' if result['data']['ativo'] else 'Inativo'}")
else:
    print(f"Erro: {result.get('error', 'Erro desconhecido')}")
```

</RequestExample>
