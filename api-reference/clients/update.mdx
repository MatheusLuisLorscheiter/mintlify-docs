---
title: "Atualizar Cliente"
api: "PUT /api/v1/clients/{id}"
description: "Atualiza os dados de um cliente existente"
---

<Tip>
Permite atualizar qualquer campo do cliente. Envie apenas os campos que deseja modificar.
</Tip>

<Warning>
Você precisa de uma API key válida no header x-api-key para usar este endpoint.
</Warning>

## Path Parameters

<ParamField path="id" type="string" required>
  ID único do cliente (UUID)
</ParamField>

## Headers

<ParamField header="Content-Type" type="string" required>
  application/json
</ParamField>

<ParamField header="x-api-key" type="string" required>
  API key de autenticação
</ParamField>

## Body

<Info>
Todos os campos são opcionais. Envie apenas os campos que deseja atualizar.
</Info>

<ParamField body="codigo" type="string">
  Código do cliente
</ParamField>

<ParamField body="nome" type="string">
  Nome completo do cliente
  
  Mínimo de 1 caractere
</ParamField>

<ParamField body="email" type="string">
  Email do cliente
  
  Deve ser um email válido
</ParamField>

<ParamField body="telefone" type="string">
  Telefone do cliente
</ParamField>

<ParamField body="cnpj_cpf" type="string">
  CNPJ ou CPF do cliente
</ParamField>

<ParamField body="endereco" type="string">
  Endereço completo
</ParamField>

<ParamField body="bairro" type="string">
  Bairro
</ParamField>

<ParamField body="cidade" type="string">
  Cidade
</ParamField>

<ParamField body="estado" type="string">
  Estado (UF)
</ParamField>

<ParamField body="observacao" type="string">
  Observações gerais sobre o cliente
</ParamField>

<ParamField body="ativo" type="boolean">
  Status do cliente
  
  - `true` - Cliente ativo
  - `false` - Cliente inativo
</ParamField>

<ParamField body="segunda" type="boolean">
  Define se o cliente atende às segundas-feiras
</ParamField>

<ParamField body="terca" type="boolean">
  Define se o cliente atende às terças-feiras
</ParamField>

<ParamField body="quarta" type="boolean">
  Define se o cliente atende às quartas-feiras
</ParamField>

<ParamField body="quinta" type="boolean">
  Define se o cliente atende às quintas-feiras
</ParamField>

<ParamField body="sexta" type="boolean">
  Define se o cliente atende às sextas-feiras
</ParamField>

<RequestExample>

```bash Atualizar Dados Básicos
curl -X PUT "https://api.sistema-representantes.com.br/api/v1/clients/123e4567-e89b-12d3-a456-426614174000" \
  -H "Content-Type: application/json" \
  -H "x-api-key: sua-api-key-aqui" \
  -d '{
    "telefone": "(11) 3456-7899",
    "email": "novo.email@supermercadocentral.com.br",
    "observacao": "Cliente VIP - Desconto especial 5%"
  }'
```

```bash Ativar/Desativar Cliente
curl -X PUT "https://api.sistema-representantes.com.br/api/v1/clients/123e4567-e89b-12d3-a456-426614174000" \
  -H "Content-Type: application/json" \
  -H "x-api-key: sua-api-key-aqui" \
  -d '{
    "ativo": false,
    "observacao": "Cliente temporariamente inativo - Reforma da loja"
  }'
```

```javascript JavaScript
// Atualizar dias de atendimento
const updateData = {
  segunda: false,
  terca: true,
  quarta: true,
  quinta: true,
  sexta: true,
  observacao: "Não atende às segundas - loja fechada"
};

const response = await fetch('https://api.sistema-representantes.com.br/api/v1/clients/123e4567-e89b-12d3-a456-426614174000', {
  method: 'PUT',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'sua-api-key-aqui'
  },
  body: JSON.stringify(updateData)
});

const result = await response.json();

if (result.success) {
  console.log('Cliente atualizado:', result.data);
} else {
  console.error('Erro:', result.error);
}
```

```python Python
import requests
import json

url = "https://api.sistema-representantes.com.br/api/v1/clients/123e4567-e89b-12d3-a456-426614174000"
headers = {
    'Content-Type': 'application/json',
    'x-api-key': 'sua-api-key-aqui'
}

# Atualizar endereço completo
data = {
    "endereco": "Avenida Paulista, 2000",
    "bairro": "Bela Vista",
    "cidade": "São Paulo",
    "estado": "SP"
}

response = requests.put(url, headers=headers, json=data)
result = response.json()

if response.status_code == 200:
    print("Cliente atualizado com sucesso!")
    print(f"Nome: {result['data']['nome']}")
    print(f"Novo endereço: {result['data']['endereco']}, {result['data']['bairro']}")
else:
    print(f"Erro: {result.get('error', 'Erro desconhecido')}")
```

## Validações

<Warning>
**Validações aplicadas:**
- Email deve ser válido se fornecido
- Nome deve ter pelo menos 1 caractere se fornecido
- Cliente deve existir no sistema
</Warning>

## Casos de Uso

<Info>
**Exemplos de uso comum:**
1. **Desativar cliente temporariamente**: Defina `ativo: false` com uma observação explicativa
2. **Atualizar dias de atendimento**: Modifique os campos de dias da semana conforme necessário
3. **Correção de dados**: Atualize informações incorretas como endereço, telefone ou email
4. **Adicionar observações**: Use o campo observação para notas importantes sobre o cliente
</Info>
