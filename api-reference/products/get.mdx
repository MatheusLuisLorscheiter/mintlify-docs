---
title: "Buscar Produto"
api: "GET /api/v1/products/{id}"
description: "Busca um produto específico pelo ID"
---

<Tip>
O ID do produto geralmente é o código/SKU definido durante a criação.
</Tip>

<Warning>
Você precisa de uma API key válida no header x-api-key para usar este endpoint.
</Warning>

## Path Parameters

<ParamField path="id" type="string" required>
  ID único do produto (código/SKU)
</ParamField>

## Headers

<ParamField header="x-api-key" type="string" required>
  API key de autenticação
</ParamField>

<RequestExample>

```bash cURL
curl -X GET "https://api.sistema-representantes.com.br/api/v1/products/QUEIJO-MUSSARELA" \
  -H "x-api-key: sua-api-key-aqui"
```

```javascript JavaScript
const response = await fetch('https://api.sistema-representantes.com.br/api/v1/products/LEITE-INTEGRAL-1L', {
  method: 'GET',
  headers: {
    'x-api-key': 'sua-api-key-aqui'
  }
});

const result = await response.json();

if (result.success) {
  console.log('Produto:', result.data);
  console.log(`Preço: R$ ${result.data.preco.toFixed(2)} por ${result.data.unidade}`);
  console.log(`Estoque: ${result.data.estoque} unidades`);
} else {
  console.error('Erro:', result.error);
}
```

```python Python
import requests

url = "https://api.sistema-representantes.com.br/api/v1/products/QUEIJO-PRATO"
headers = {
    'x-api-key': 'sua-api-key-aqui'
}

response = requests.get(url, headers=headers)
result = response.json()

if response.status_code == 200:
    produto = result['data']
    print(f"Produto: {produto['nome']}")
    print(f"Descrição: {produto['descricao']}")
    print(f"Preço: R$ {produto['preco']:.2f} por {produto['unidade']}")
    print(f"Estoque: {produto['estoque']} unidades")
    
    if produto['estoque'] < produto['estoque_minimo']:
        print("⚠️ ALERTA: Estoque abaixo do mínimo!")
else:
    print(f"Erro: {result.get('error', 'Erro desconhecido')}")
```

</RequestExample>

## Response

### Sucesso

<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="data" type="object">
  Dados completos do produto
  
  <Expandable title="Estrutura do produto">
    <ResponseField name="id" type="string">
      ID único do produto (código/SKU)
    </ResponseField>
    <ResponseField name="nome" type="string">
      Nome do produto
    </ResponseField>
    <ResponseField name="descricao" type="string">
      Descrição detalhada
    </ResponseField>
    <ResponseField name="preco" type="number">
      Preço unitário
    </ResponseField>
    <ResponseField name="unidade" type="string">
      Unidade de medida
    </ResponseField>
    <ResponseField name="estoque" type="number">
      Quantidade atual em estoque
    </ResponseField>
    <ResponseField name="estoque_minimo" type="number">
      Estoque mínimo para alerta
    </ResponseField>
    <ResponseField name="ativo" type="boolean">
      Status do produto
    </ResponseField>
    <ResponseField name="created_at" type="string">
      Data de criação
    </ResponseField>
    <ResponseField name="updated_at" type="string">
      Data da última atualização
    </ResponseField>
  </Expandable>
</ResponseField>

## Informações Úteis

<Info>
**Verificações de estoque:**
- Se `estoque < estoque_minimo`, o produto precisa de reposição
- Se `estoque = 0`, o produto está sem estoque
- Use estas informações para alertas e notificações
</Info>
