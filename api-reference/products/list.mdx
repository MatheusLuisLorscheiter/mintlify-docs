---
title: "Listar Produtos"
api: "GET /api/v1/products"
description: "Lista todos os produtos cadastrados com paginação"
---

<Tip>
Use os parâmetros de busca e filtro para encontrar produtos específicos.
</Tip>

<Warning>
Você precisa de uma API key válida no header x-api-key para usar este endpoint.
</Warning>

## Headers

<ParamField header="x-api-key" type="string" required>
  API key de autenticação
</ParamField>

## Query Parameters

<ParamField query="page" type="number" default="1">
  Número da página para paginação
</ParamField>

<ParamField query="limit" type="number" default="20">
  Quantidade de itens por página (máximo 100)
</ParamField>

<ParamField query="search" type="string">
  Busca por ID, nome ou descrição do produto
</ParamField>

<ParamField query="ativo" type="boolean">
  Filtrar produtos por status
  - `true` - Apenas produtos ativos
  - `false` - Apenas produtos inativos
  - Omitir para listar todos
</ParamField>

<RequestExample>

```bash cURL
curl -X GET "https://api.sistema-representantes.com.br/api/v1/products?page=1&limit=10&ativo=true" \
  -H "x-api-key: sua-api-key-aqui"
```

```javascript JavaScript
const response = await fetch('https://api.sistema-representantes.com.br/api/v1/products?search=queijo', {
  method: 'GET',
  headers: {
    'x-api-key': 'sua-api-key-aqui'
  }
});

const result = await response.json();

if (result.success) {
  console.log('Total de produtos:', result.total);
  result.data.forEach(produto => {
    console.log(`${produto.id}: ${produto.nome} - R$ ${produto.preco}`);
  });
}
```

```python Python
import requests

url = "https://api.sistema-representantes.com.br/api/v1/products"
headers = {
    'x-api-key': 'sua-api-key-aqui'
}
params = {
    'page': 1,
    'limit': 50,
    'ativo': 'true'
}

response = requests.get(url, headers=headers, params=params)
result = response.json()

if response.status_code == 200:
    print(f"Total de produtos ativos: {result['total']}")
    for produto in result['data']:
        print(f"- {produto['nome']}: R$ {produto['preco']:.2f} ({produto['unidade']})")
else:
    print(f"Erro: {result.get('error', 'Erro desconhecido')}")
```

</RequestExample>
