---
title: "Atualizar Produto"
api: "PUT /api/v1/products/{id}"
description: "Atualiza os dados de um produto existente"
---

<Tip>
Use este endpoint para atualizar preços, informações ou status de produtos.
</Tip>

<Warning>
Você precisa de uma API key válida no header x-api-key para usar este endpoint.
</Warning>

## Path Parameters

<ParamField path="id" type="string" required>
  ID único do produto (código/SKU)
</ParamField>

## Headers

<ParamField header="Content-Type" type="string" required>
  application/json
</ParamField>

<ParamField header="x-api-key" type="string" required>
  API key de autenticação
</ParamField>

## Body

<Info>
Todos os campos são opcionais. Envie apenas os campos que deseja atualizar.
</Info>

<ParamField body="nome" type="string">
  Nome do produto
  
  Mínimo de 1 caractere
</ParamField>

<ParamField body="descricao" type="string">
  Descrição detalhada do produto
  
  Pode ser nulo para remover a descrição
</ParamField>

<ParamField body="preco" type="number">
  Preço unitário do produto
  
  Deve ser maior ou igual a 0
</ParamField>



<ParamField body="ativo" type="boolean">
  Status do produto
  
  - `true` - Produto ativo
  - `false` - Produto inativo
</ParamField>

<RequestExample>

```bash Atualizar Preço
curl -X PUT "https://api.sistema-representantes.com.br/api/v1/products/QUEIJO-MUSSARELA" \
  -H "Content-Type: application/json" \
  -H "x-api-key: sua-api-key-aqui" \
  -d '{
    "preco": 47.90
  }'
```

```javascript JavaScript
// Desativar produto
const response = await fetch('https://api.sistema-representantes.com.br/api/v1/products/PROD-DESCONTINUADO', {

    'Content-Type': 'application/json',
```

```python Python
import requests
import json

url = "https://api.sistema-representantes.com.br/api/v1/products/QUEIJO-PRATO"
headers = {
    'Content-Type': 'application/json',
    'x-api-key': 'sua-api-key-aqui'
}

# Atualização completa após reajuste de preços
data = {
    "preco": 44.90,

    "descricao": "Queijo prato fatiado premium, embalagem de 1kg - NOVO PREÇO"
}

response = requests.put(url, headers=headers, json=data)
result = response.json()

if response.status_code == 200:
    print("Produto atualizado com sucesso!")
    print(f"Nome: {result['data']['nome']}")
    print(f"Novo preço: R$ {result['data']['preco']:.2f}")

else:
    print(f"Erro: {result.get('error', 'Erro desconhecido')}")
    if 'details' in result:
        print("Detalhes:", result['details'])
```

## Validações

<Warning>
**Validações aplicadas:**
- Nome deve ter pelo menos 1 caractere se fornecido
- Preço deve ser maior ou igual a 0 se fornecido

- Produto deve existir no sistema
</Warning>

## Casos de Uso Comuns

<Info>
**Exemplos práticos:**

1. **Reajuste de preços**: Atualize apenas o campo `preco`
2. **Correção de dados**: Atualize descrição ou unidade de medida
3. **Descontinuar produto**: Defina `ativo: false` e atualize a descrição
4. **Mudança de unidade**: Altere a unidade de medida quando necessário
5. **Correção de informações**: Atualize nome ou descrição incorretos
</Info>
