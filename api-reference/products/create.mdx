---
title: "Criar Produto"
api: "POST /api/v1/products"
description: "Cadastra um novo produto no sistema"
---

<Tip>
O ID do produto é definido pelo usuário e geralmente representa o código/SKU do produto.
</Tip>

<Warning>
Você precisa de uma API key válida no header x-api-key para usar este endpoint.
</Warning>

## Headers

<ParamField header="Content-Type" type="string" required>
  application/json
</ParamField>

<ParamField header="x-api-key" type="string" required>
  API key de autenticação
</ParamField>

## Body

<ParamField body="id" type="string" required>
  ID único do produto (código/SKU)
  
  Este será o identificador do produto no sistema
</ParamField>

<ParamField body="nome" type="string" required>
  Nome do produto
  
  Mínimo de 1 caractere
</ParamField>

<ParamField body="descricao" type="string">
  Descrição detalhada do produto
</ParamField>

<ParamField body="preco" type="number" required>
  Preço unitário do produto
  
  Deve ser maior ou igual a 0
</ParamField>


<ParamField body="ativo" type="boolean" default="true">
  Status inicial do produto
  
  - `true` - Produto ativo
  - `false` - Produto inativo
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://api.sistema-representantes.com.br/api/v1/products" \
  -H "Content-Type: application/json" \
  -H "x-api-key: sua-api-key-aqui" \
  -d '{
    "id": "QUEIJO-PRATO",
    "nome": "Queijo Prato",
    "descricao": "Queijo prato fatiado, embalagem de 1kg",
    "preco": 42.50,


    "ativo": true
  }'
```

```javascript JavaScript
const produto = {
  id: "REQUEIJAO-200G",
  nome: "Requeijão Cremoso 200g",
  descricao: "Requeijão cremoso tradicional, copo de 200g",
  preco: 8.90,

const response = await fetch('https://api.sistema-representantes.com.br/api/v1/products', {

    'Content-Type': 'application/json',

const result = await response.json();

```

```python Python
import requests
import json

url = "https://api.sistema-representantes.com.br/api/v1/products"
headers = {
    'Content-Type': 'application/json',
    'x-api-key': 'sua-api-key-aqui'
}

# Criar produto em caixa
produto = {
    "id": "IOGURTE-MORANGO-CX",
    "nome": "Iogurte Sabor Morango",
    "descricao": "Iogurte integral sabor morango, caixa com 12 unidades de 170g",
    "preco": 36.00,


    "ativo": True
}

response = requests.post(url, headers=headers, json=produto)
result = response.json()

if response.status_code == 201:
    print("Produto criado com sucesso!")
    print(f"ID: {result['data']['id']}")
    print(f"Nome: {result['data']['nome']}")
    print(f"Preço: R$ {result['data']['preco']:.2f} por {result['data']['unidade']}")
else:
    print(f"Erro: {result.get('error', 'Erro desconhecido')}")
```

## Validações

<Warning>
**Validações aplicadas:**
- ID deve ser único no sistema
- Nome deve ter pelo menos 1 caractere
- Preço deve ser maior ou igual a 0

</Warning>

## Dicas

<Tip>
**Boas práticas para IDs de produtos:**
- Use códigos significativos (ex: PROD-001, QUEIJO-MUSSARELA)
- Mantenha um padrão consistente
- Evite caracteres especiais
- Use maiúsculas para facilitar busca
</Tip>
