---
title: "Listar Representadas"
api: "GET /api/v1/representadas"
description: "Lista todas as representadas cadastradas com paginação"
---

<Tip>
Use os parâmetros de busca e filtro para encontrar representadas específicas.
</Tip>

<Warning>
Você precisa de uma API key válida no header x-api-key para usar este endpoint.
</Warning>

## Headers

<ParamField header="x-api-key" type="string" required>
  API key de autenticação
</ParamField>

## Query Parameters

<ParamField query="page" type="number" default="1">
  Número da página para paginação
</ParamField>

<ParamField query="limit" type="number" default="20">
  Quantidade de itens por página (máximo 100)
</ParamField>

<ParamField query="search" type="string">
  Busca por nome, CNPJ ou cidade da representada
</ParamField>

<ParamField query="ativo" type="boolean">
  Filtrar representadas por status
  - `true` - Apenas representadas ativas
  - `false` - Apenas representadas inativas
  - Omitir para listar todas
</ParamField>

<RequestExample>

```bash cURL
curl -X GET "https://api.sistema-representantes.com.br/api/v1/representadas?page=1&limit=10&ativo=true" \
  -H "x-api-key: sua-api-key-aqui"
```

```javascript JavaScript
const response = await fetch('https://api.sistema-representantes.com.br/api/v1/representadas?search=laticínios', {
  method: 'GET',
  headers: {
    'x-api-key': 'sua-api-key-aqui'
  }
});

const result = await response.json();

if (result.success) {
  console.log('Total de representadas:', result.total);
  result.data.forEach(rep => {
    console.log(`${rep.nome} - ${rep.cidade}/${rep.estado} - Comissão: ${rep.comissao}%`);
  });
}
```

```python Python
import requests

url = "https://api.sistema-representantes.com.br/api/v1/representadas"
headers = {
    'x-api-key': 'sua-api-key-aqui'
}
params = {
    'page': 1,
    'limit': 50,
    'ativo': 'true'
}

response = requests.get(url, headers=headers, params=params)
result = response.json()

if response.status_code == 200:
    print(f"Total de representadas ativas: {result['total']}")
    for rep in result['data']:
        print(f"- {rep['nome']} ({rep['cnpj']})")
        print(f"  Cidade: {rep['cidade']}/{rep['estado']}")
        print(f"  Comissão: {rep['comissao']}%")
        print()
else:
    print(f"Erro: {result.get('error', 'Erro desconhecido')}")
```

</RequestExample>
