---
title: "Buscar Representada"
api: "GET /api/v1/representadas/{id}"
description: "Busca uma representada específica pelo ID com estatísticas de vendas"
---

<Tip>
Este endpoint retorna informações detalhadas da representada incluindo estatísticas de vendas.
</Tip>

<Warning>
Você precisa de uma API key válida no header x-api-key para usar este endpoint.
</Warning>

## Path Parameters

<ParamField path="id" type="string" required>
  ID único da representada (UUID)
</ParamField>

## Headers

<ParamField header="x-api-key" type="string" required>
  API key de autenticação
</ParamField>

<RequestExample>

```bash cURL
curl -X GET "https://api.sistema-representantes.com.br/api/v1/representadas/987e6543-e89b-12d3-a456-426614174999" \
  -H "x-api-key: sua-api-key-aqui"
```

```javascript JavaScript
const response = await fetch('https://api.sistema-representantes.com.br/api/v1/representadas/987e6543-e89b-12d3-a456-426614174999', {
  method: 'GET',
  headers: {
    'x-api-key': 'sua-api-key-aqui'
  }
});

const result = await response.json();

if (result.success) {
  console.log('Representada:', result.data.nome);
  console.log('Estatísticas:');
  console.log(`- Total de vendas: ${result.data.estatisticas.total_vendas}`);
  console.log(`- Total de clientes: ${result.data.estatisticas.total_clientes}`);
  console.log(`- Valor total vendido: R$ ${result.data.estatisticas.valor_total.toFixed(2)}`);
} else {
  console.error('Erro:', result.error);
}
```

```python Python
import requests

url = "https://api.sistema-representantes.com.br/api/v1/representadas/987e6543-e89b-12d3-a456-426614174999"
headers = {
    'x-api-key': 'sua-api-key-aqui'
}

response = requests.get(url, headers=headers)
result = response.json()

if response.status_code == 200:
    rep = result['data']
    stats = rep['estatisticas']
    
    print(f"Representada: {rep['nome']}")
    print(f"CNPJ: {rep['cnpj']}")
    print(f"Cidade: {rep['cidade']}/{rep['estado']}")
    print(f"Comissão: {rep['comissao']}%")
    print("\nEstatísticas de Vendas:")
    print(f"- Total de vendas: {stats['total_vendas']}")
    print(f"- Clientes atendidos: {stats['total_clientes']}")
    print(f"- Valor total: R$ {stats['valor_total']:,.2f}")
    
    if stats['total_vendas'] > 0:
        ticket_medio = stats['valor_total'] / stats['total_vendas']
        print(f"- Ticket médio: R$ {ticket_medio:,.2f}")
else:
    print(f"Erro: {result.get('error', 'Erro desconhecido')}")
```

## Informações Úteis

<Info>
**As estatísticas incluem:**
- Total de vendas realizadas para esta representada
- Quantidade de clientes únicos que compraram produtos desta representada
- Valor total acumulado de todas as vendas

Use estas informações para:
- Análise de desempenho
- Cálculo de comissões
- Tomada de decisão sobre parcerias
- Identificação de representadas mais lucrativas
</Info>
