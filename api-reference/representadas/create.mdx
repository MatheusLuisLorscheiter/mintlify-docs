---
title: "Criar Representada"
api: "POST /api/v1/representadas"
description: "Cadastra uma nova representada no sistema"
---

<Tip>
O sistema gera automaticamente um ID único (UUID) para cada representada criada.
</Tip>

<Warning>
Você precisa de uma API key válida no header x-api-key para usar este endpoint.
</Warning>

## Headers

<ParamField header="Content-Type" type="string" required>
  application/json
</ParamField>

<ParamField header="x-api-key" type="string" required>
  API key de autenticação
</ParamField>

## Body

<ParamField body="nome" type="string" required>
  Nome/Razão social da representada
  
  Mínimo de 1 caractere
</ParamField>

<ParamField body="cnpj" type="string">
  CNPJ da representada
  
  Formato: XX.XXX.XXX/XXXX-XX
</ParamField>

<ParamField body="email" type="string">
  Email de contato
  
  Deve ser um email válido
</ParamField>

<ParamField body="telefone" type="string">
  Telefone de contato
</ParamField>

<ParamField body="endereco" type="string">
  Endereço completo da representada
</ParamField>

<ParamField body="cidade" type="string">
  Cidade
</ParamField>

<ParamField body="estado" type="string">
  Estado (UF)
  
  Exemplo: SP, RJ, MG
</ParamField>

<ParamField body="contato" type="string">
  Nome da pessoa de contato na representada
</ParamField>

<ParamField body="observacao" type="string">
  Observações gerais sobre a representada
</ParamField>


<ParamField body="ativo" type="boolean" default="true">
  Status inicial da representada
  
  - `true` - Representada ativa
  - `false` - Representada inativa
</ParamField>

```python Python
import requests
import json

url = "https://api.sistema-representantes.com.br/api/v1/representadas"
headers = {
    'Content-Type': 'application/json',
    'x-api-key': 'sua-api-key-aqui'
}

representada = {
    "nome": "Frios Premium Distribuidora",
    "cnpj": "45.678.901/0001-23",
    "email": "contato@frios-premium.com",
    "telefone": "(31) 3456-7890",
    "endereco": "Rua do Comércio, 1500",
    "cidade": "Belo Horizonte",
    "estado": "MG",
    "contato": "Carlos Mendes",
    "observacao": "Especializada em frios e embutidos premium",
    "comissao": 8.0,
    "ativo": True
}

response = requests.post(url, headers=headers, json=representada)
result = response.json()

if response.status_code == 201:
    print("Representada criada com sucesso!")
    print(f"ID: {result['data']['id']}")
    print(f"Nome: {result['data']['nome']}")
    print(f"Comissão: {result['data']['comissao']}%")
else:
    print(f"Erro: {result.get('error', 'Erro desconhecido')}")
    if 'details' in result:
        print("Detalhes:", result['details'])
```

## Validações

<Warning>
**Validações aplicadas:**
- Nome deve ter pelo menos 1 caractere
- Email deve ser válido se fornecido
- Comissão deve estar entre 0 e 100
</Warning>

## Dicas

<Tip>
**Boas práticas:**
- Mantenha o CNPJ no formato padrão brasileiro
- Use a comissão padrão que será aplicada nas vendas
- Preencha o campo contato para facilitar a comunicação
- Use o campo observação para informações importantes sobre prazo de pagamento, condições especiais, etc.
</Tip>
