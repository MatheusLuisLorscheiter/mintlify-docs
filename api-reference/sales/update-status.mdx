---
title: "Atualizar Status da Venda"
api: "PATCH /api/v1/sales/{id}/status"
description: "Atualiza apenas o status de uma venda existente"
---

<Tip>
Use este endpoint para alterar rapidamente o status de uma venda sem precisar enviar todos os dados.
</Tip>

<Warning>
Você precisa de uma API key válida no header x-api-key para usar este endpoint.
</Warning>

## Path Parameters

<ParamField path="id" type="string" required>
  ID único da venda (UUID)
</ParamField>

## Headers

<ParamField header="Content-Type" type="string" required>
  application/json
</ParamField>

<ParamField header="x-api-key" type="string" required>
  API key de autenticação
</ParamField>

## Body

<ParamField body="status" type="string" required>
  Novo status da venda
  
  Valores aceitos:
  - `PAGA` - Venda paga
  - `AGUARDANDO PAGAMENTO` - Aguardando pagamento
  - `CANCELADA` - Venda cancelada
  - `ENVIADA` - Venda enviada
  - `ENTREGUE` - Venda entregue
</ParamField>

```python Python
import requests
import json

url = "https://api.sistema-representantes.com.br/api/v1/sales/789e0123-e89b-12d3-a456-426614174222/status"
headers = {
    'Content-Type': 'application/json',
    'x-api-key': 'sua-api-key-aqui'
}

# Marcar como enviada
data = {
    "status": "ENVIADA"
}

response = requests.patch(url, headers=headers, json=data)
result = response.json()

if response.status_code == 200:
    print("Status atualizado com sucesso!")
    print(f"Novo status: {result['data']['status']}")
    print(f"Mensagem: {result['message']}")
else:
    print(f"Erro: {result.get('error', 'Erro desconhecido')}")
```

```php PHP
<?php
$url = 'https://api.sistema-representantes.com.br/api/v1/sales/789e0123-e89b-12d3-a456-426614174222/status';
$headers = [
    'Content-Type: application/json',
    'x-api-key: sua-api-key-aqui'

    'status' => 'AGUARDANDO PAGAMENTO'
];

$ch = curl_init($url);
curl_setopt($ch, CURLOPT_CUSTOMREQUEST, "PATCH");
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
curl_setopt($ch, CURLOPT_HTTPHEADER, $headers);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$httpCode = curl_getinfo($ch, CURLINFO_HTTP_CODE);
curl_close($ch);

$result = json_decode($response, true);

if ($httpCode === 200) {
    echo "Status atualizado com sucesso!\n";
    echo "Novo status: " . $result['data']['status'] . "\n";
    echo $result['message'] . "\n";
} else {
    echo "Erro: " . $result['error'] . "\n";
}
?>
```

## Fluxo de Status

<Info>
**Fluxo típico de status de uma venda:**

1. **PAGA** → Status inicial padrão quando a venda é criada
2. **AGUARDANDO PAGAMENTO** → Se o pagamento ainda não foi confirmado
3. **ENVIADA** → Quando a mercadoria foi despachada
4. **ENTREGUE** → Quando a mercadoria foi recebida pelo cliente
5. **CANCELADA** → Se a venda foi cancelada por qualquer motivo

Você pode alterar de qualquer status para qualquer outro status.
</Info>

## Casos de Uso

<Tip>
**Exemplos práticos:**

- **Confirmar pagamento**: Mudar de `AGUARDANDO PAGAMENTO` para `PAGA`
- **Despachar mercadoria**: Mudar de `PAGA` para `ENVIADA`
- **Confirmar entrega**: Mudar de `ENVIADA` para `ENTREGUE`
- **Cancelar venda**: Mudar para `CANCELADA` de qualquer status
- **Reverter cancelamento**: Mudar de `CANCELADA` para o status apropriado
</Tip>
