---
title: "Buscar Venda"
api: "GET /api/v1/sales/{id}"
description: "Busca uma venda específica pelo ID"
---

<Tip>
Retorna todos os detalhes de uma venda, incluindo informações do cliente, representada e todos os itens.
</Tip>

<Warning>
Você precisa de uma API key válida no header x-api-key para usar este endpoint.
</Warning>

## Path Parameters

<ParamField path="id" type="string" required>
  ID único da venda (UUID)
</ParamField>

## Headers

<ParamField header="x-api-key" type="string" required>
  API key de autenticação
</ParamField>

<RequestExample>

```bash cURL
curl -X GET "https://api.sistema-representantes.com.br/api/v1/sales/789e0123-e89b-12d3-a456-426614174222" \
  -H "x-api-key: sua-api-key-aqui"
```

```javascript JavaScript
const response = await fetch('https://api.sistema-representantes.com.br/api/v1/sales/789e0123-e89b-12d3-a456-426614174222', {
  method: 'GET',
  headers: {
    'x-api-key': 'sua-api-key-aqui'
  }
});

const result = await response.json();

if (result.success) {
  console.log('Venda encontrada:', result.data);
  console.log('Itens:', result.data.itens);
} else {
  console.error('Erro:', result.error);
}
```

```python Python
import requests

url = "https://api.sistema-representantes.com.br/api/v1/sales/789e0123-e89b-12d3-a456-426614174222"
headers = {
    'x-api-key': 'sua-api-key-aqui'
}

response = requests.get(url, headers=headers)
result = response.json()

if response.status_code == 200:
    print(f"Venda: {result['data']['pedido']}")
    print(f"Cliente: {result['data']['cliente_nome']}")
    print(f"Valor Total: R$ {result['data']['valor_total']:.2f}")
    print("\nItens:")
    for item in result['data']['itens']:
        print(f"- {item['produto_id']}: {item['quantidade']} x R$ {item['valor_unitario']:.2f}")
else:
    print(f"Erro: {result.get('error', 'Erro desconhecido')}")
```

</RequestExample>
