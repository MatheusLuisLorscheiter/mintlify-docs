## Início Rápido

<Steps>
  <Step title="Obtenha sua API Key">
    Acesse o painel administrativo e gere sua chave de autenticação
  </Step>
  <Step title="Configure a autenticação">
    Adicione o header `x-api-key` em todas as suas requisições
  </Step>
  <Step title="Faça sua primeira requisição">
    Teste a API listando seus clientes cadastrados
  </Step>
</Steps>

```bash
curl -X GET https://api.sistema-representantes.com.br/api/v1/clients \
  -H "x-api-key: sua-api-key-aqui"
```

## Base URL

Todas as requisições devem ser feitas para:

```
https://api.sistema-representantes.com.br/api/v1
```

## Recursos da API

### 🏢 Clientes
- Listar clientes com paginação e busca
- Criar novos clientes com validação automática
- Gerenciar informações completas (CNPJ/CPF, endereço, contatos)

### 💰 Vendas
- Registrar vendas com múltiplos produtos
- Cálculo automático de valores
- Controle de status (Pedido, Faturado, Enviado, Entregue)
- Geração automática de códigos de pedido
