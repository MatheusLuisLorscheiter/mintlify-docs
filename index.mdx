# Sistema de Representantes - API REST

Bem-vindo à documentação oficial da API REST do Sistema de Representantes. Esta API permite integração com sistemas externos para gerenciar clientes e vendas.

## Visão Geral

O Sistema de Representantes oferece endpoints REST para:

- **Gerenciamento de Clientes**: Criar e listar clientes
- **Gerenciamento de Vendas**: Criar e listar vendas com itens

## Começando

Para começar a usar a API, você precisa:

1. **Obter uma API Key**: Entre em contato com o administrador do sistema
2. **Configurar a autenticação**: Use o header `x-api-key` em todas as requisições
3. **Fazer sua primeira requisição**: Teste com o endpoint de listagem de clientes

## Base URL

```
https://api.sistema-representantes.com.br/api/v1
```

## Autenticação

Todas as requisições devem incluir o header de autenticação:

```bash
x-api-key: sua-chave-api-aqui
```

## Códigos de Status HTTP

| Código | Descrição |
|--------|-----------|
| 200    | Sucesso |
| 201    | Recurso criado com sucesso |
| 400    | Requisição inválida |
| 401    | Não autorizado (API key inválida ou ausente) |
| 403    | Acesso negado (empresa inativa) |
| 500    | Erro interno do servidor |
