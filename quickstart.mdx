# Início Rápido

Este guia mostra como fazer suas primeiras requisições à API.

## 1. Obtenha sua API Key

<Steps>
  <Step title="Acesse o painel">
    Entre no sistema com suas credenciais
  </Step>
  <Step title="Navegue até API Keys">
    Vá em **Configurações** → **Integrações** → **API Keys**
  </Step>
  <Step title="Gere uma nova chave">
    Clique em **Gerar Nova API Key** e copie a chave gerada
  </Step>
</Steps>

## 2. Configure o ambiente

Salve sua API key em uma variável de ambiente:

```bash
export API_KEY="sua-api-key-aqui"
export API_URL="https://api.sistema-representantes.com.br/api/v1"
```

## 3. Teste a conexão

Liste seus clientes para verificar se tudo está funcionando:

```bash
curl -X GET "$API_URL/clients" \
  -H "x-api-key: $API_KEY"
```

## 4. Exemplos práticos

### Criar um cliente

```bash
curl -X POST "$API_URL/clients" \
  -H "x-api-key: $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "nome": "Novo Cliente Ltda",
    "email": "contato@novocliente.com",
    "telefone": "(11) 98765-4321",
    "cidade": "São Paulo",
    "estado": "SP"
  }'
```

### Registrar uma venda

```bash
curl -X POST "$API_URL/sales" \
  -H "x-api-key: $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "clienteId": "id-do-cliente",
    "representadaId": "id-da-representada",
    "status": "Pedido",
    "itens": [
      {
        "produtoId": "PROD-001",
        "quantidade": 10,
        "valorUnitario": 50.00
      }
    ]
  }'
```
